## Identifying Firm Users

Follow the steps below to identify Firm User roles for R4P (Wholesale) accounts. This article explains what each role means and how their access levels differ. It will also help you quickly understand permission levels and solve access-related issues.

In RUN, go to Companies > Settings > Security > Manage user security.

In the Role field, you can see the user's role. 

Refer to the table below to see the available user roles for Firm Users.

| Users | Roles |
| - | - |
| Firm | Can access all payroll - related functions Can add, change, or remove other users and company groups Can access Timekeeping Plus |
| Firm Limited Payroll (PR) & Employee (EE) Entry | Limited access to multiple clients Can enter payroll Can access Timekeeping Plus |
| Firm Limited Employee (EE) Entry | Limited access to multiple clients Can enter payroll Can access Timekeeping Plus |
| Firm Limited Employee (EE) & Payroll (PR) Process | Access to multiple clients Can enter and run payroll Is able to be a Firm AND Client user Can access Timekeeping Plus |

## Identifying Client Users 

Follow the steps below to identify Client User roles for R4P (Wholesale) accounts. This article explains what each role means and how their access levels differ. It will also help you quickly understand permission levels and solve access-related issues

In RUN, go to Companies. 

Click on the company name. 

> **[Image: lsa1770833429187](images/lsa1770833429187.png)**
> This image shows a web interface listing companies, likely from a payroll or business management platform. The page is titled 'Companies' and provides a searchable and filterable list of companies with details such as company name/ID, company details, contact person, payroll due date, and next check date. Each company entry includes a clickable name, client ID, company type, contact name and phone number, and relevant dates. There is a search bar and a 'Filter' button at the top for refining the list.

Click Settings > Company > RUN access. 

In the User access role field, you can see the user's role. 

Refer to the table below to view available user roles for Client Users.

| Users | Roles |
| - | - |
| Client View Only | Can only VIEW the Home page, employee's, and reports Cannot access Payroll Home Cannot access Timekeeping Plus |
| Client Payroll and Employee (EE) Entry | Can access the Home page, employee's, and reports Can access the Home page Can access Timekeeping Plus |
| Client Payroll Entry | Can access reports, General Ledger (GL), entering payroll Can access Timekeeping Plus |
| Time Only | Can ONLY access Timekeeping Plus |
