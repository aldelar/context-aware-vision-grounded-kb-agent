Identifying RUN for Partners (Wholesale) User Roles in RUN

  February 11, 2026

Identifying Firm Users

Follow the steps below to identify Firm User roles for R4P (Wholesale) accounts. This article explains what each role means and how their access levels differ. It will also help you quickly understand permission levels and solve access-related issues.

  1. In RUN, go to Companies > Settings > Security > Manage user security.

  2. In the Role field, you can see the user's role.
Refer to the table below to see the available user roles for Firm Users.
Users Roles
Firm
  • Can access all payroll - related functions
  • Can add, change, or remove other users and company groups
  • Can access Timekeeping Plus
Firm Limited Payroll (PR) & Employee (EE) Entry
  • Limited access to multiple clients
  • Can enter payroll
  • Can access Timekeeping Plus
Firm Limited Employee (EE) Entry
  • Limited access to multiple clients
  • Can enter payroll
  • Can access Timekeeping Plus
Firm Limited Employee (EE) & Payroll (PR) Process
  • Access to multiple clients
  • Can enter and run payroll
  • Is able to be a Firm AND Client user
  • Can access Timekeeping Plus

Identifying Client Users

Follow the steps below to identify Client User roles for R4P (Wholesale) accounts. This article explains what each role means and how their access levels differ. It will also help you quickly understand permission levels and solve access-related issues

  1. In RUN, go to Companies.
  2. Click on the company name.

  3. Click Settings > Company > RUN access.
  4. In the User access role field, you can see the user's role.
Refer to the table below to view available user roles for Client Users.
Users Roles
Client View Only
  • Can only VIEW the Home page, employee's, and reports
  • Cannot access Payroll Home
  • Cannot access Timekeeping Plus
Client Payroll and Employee (EE) Entry
  • Can access the Home page, employee's, and reports
  • Can access the Home page
  • Can access Timekeeping Plus
Client Payroll Entry
  • Can access reports, General Ledger (GL), entering payroll
  • Can access Timekeeping Plus
Time Only Can ONLY access Timekeeping Plus
